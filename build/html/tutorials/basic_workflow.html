Basic Workflow#

This tutorial walks through a standard single-cell analysis pipeline: feature selection, normalization, dimensionality reduction, clustering, and marker gene identification. It starts with steps from the Loading and Quality Control tutorial using the same Parse Biosciences ~10 million cell PBMC dataset.

Setup#

import polars as pl

print(f'{os.cpu_count()} CPUs, '

Feature selection#

hvg() selects highly variable genes using Seurat’s variance-stabilization approach. It operates on raw counts and must be run before normalize(). By default, it selects the top 2,000 genes.

When your data has multiple batches, pass batch_column to identify HVGs that are variable across batches rather than just within one:

with Timer('HVG'):
    sc = sc.hvg(batch_column='donor')

HVG...
HVG took 2s 280ms

This adds a highly_variable Boolean column and a highly_variable_rank integer column to var. Downstream methods like pca() automatically use only the highly variable genes.

Normalization#

normalize() log-transforms the counts. The default method, PFlog1pPF (Booeshaghi et al. 2022), applies proportional fitting before and after log1p to correct for composition bias. With method='logCP10k', it matches Seurat’s NormalizeData().

with Timer('Normalize'):
    sc = sc.normalize()

Normalize...
Normalize took 27s 760ms

PCA#

pca() computes principal components from the normalized, highly variable genes. The default is 50 PCs, and can be adjusted with num_PCs.

with Timer('PCA'):
    sc = sc.pca()

PCA...
PCA took 20s 630ms

PCs are stored in obsm['pca'] as a NumPy array of shape (n_cells, 50).

Nearest neighbors#

neighbors() computes the 20 nearest neighbors of each cell in PC space, using a fast approximate search based on FAISS’s IndexIVFFlat strategy. shared_neighbors() then builds a shared nearest neighbor (SNN) graph — two cells are connected in proportion to how many neighbors they share. This is the same approach used by Seurat.

with Timer('Neighbors'):
    sc = sc.neighbors().shared_neighbors()

Neighbors...
Neighbors took 58s 570ms

The nearest-neighbor indices, distances, and SNN graph are stored in obsm['neighbors'], obsm['distances'], and obsm['shared_neighbors'].

Note

If you subset your data after computing neighbors (e.g. via filter_obs()), the neighbor graph becomes invalid and must be recomputed. brisc enforces this and will raise an error if you try to use stale neighbors.

Batch integration#

If your dataset contains batch effects, harmonize() corrects the PCs using Harmony. Run it after pca() and before neighbors():

sc = sc.harmonize(batch_column='donor')
# downstream steps will use obsm['harmony'] instead of obsm['pca']
sc = sc.neighbors(PC_key='harmony').shared_neighbors()

brisc’s Harmony implementation is parallelized via a nested block strategy and uses a k-means variant called k-means|| for initialization.

Clustering#

cluster() runs Leiden clustering on the SNN graph. The resolution parameter controls granularity — higher values produce more clusters. You can pass multiple resolutions to evaluate them in parallel:

with Timer('Cluster'):
    sc = sc.cluster(resolution=[0.25, 0.5, 1.0, 1.5, 2.0])

Cluster...
Cluster took 5m 21s

Each resolution adds a column to obs: cluster_0 through cluster_4 (or a custom name via cluster_column). The implementation is based on GVE-Leiden.

Embedding#

brisc offers three embedding methods for visualization, all of which operate on the PCs and neighbor graph:

  • pacmap() — PaCMAP, captures global structure well. Default choice.

  • localmap() — LocalMAP, balances local and global structure.

  • umap() — the standard UMAP algorithm.

with Timer('PaCMAP'):
    sc = sc.pacmap()

PaCMAP...
PaCMAP took 1m 8s

Embeddings are stored as 2-column NumPy arrays in obsm (e.g. obsm['pacmap']). Visualize with plot_embedding():

sc.plot_embedding('cell_type', embedding_key='pacmap')

Marker genes#

find_markers() identifies genes that distinguish each cell type from the rest, using an adaptation of Fischer and Gillis 2021. It works on binarized expression (detected vs. not detected), so it gives the same result before or after normalization.

Genes must have a detection rate of at least 25% in the target cell type and at least a 2-fold enrichment relative to every other cell type. Pareto-optimal genes — those not dominated by any other gene on both metrics — are flagged.

with Timer('Markers'):
    markers = sc.find_markers('cell_type')

Pipeline summary#

The full pipeline chains together in ~10 minutes on 10 million cells:

sc = (
    SingleCell('data.h5ad', num_threads=-1)
    .qc(remove_doublets=True, batch_column='sample',
        subset=False, allow_float=True)
    .hvg(batch_column='donor')
    .normalize()
    .pca()
    .neighbors()
    .shared_neighbors()
    .cluster(resolution=[0.25, 0.5, 1.0, 1.5, 2.0])
    .pacmap()
)
markers = sc.find_markers('cell_type')
